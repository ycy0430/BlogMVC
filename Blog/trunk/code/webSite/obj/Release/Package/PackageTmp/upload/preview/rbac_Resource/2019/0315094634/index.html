关于许浩杰同学参与

“市政工程项目大数据平台”的评价

 

 

该项目是王志瑞老师和云宝网络有限公司签署的委托开发合同，合同第三方是常州照明管理处，该项目主要完成常州照明管理处照明施工项目的管理与控制，主要包括：项目的签订，劳务合同的签订，项目施工管理，项目竣工验收，项目移交，项目结算等相关的内容。

许浩杰同学从大二就跟着王志瑞老师参与兴趣小组学习，正好参与到该项目中，许浩杰同学从项目的需求调研，到项目的开发，最后到项目的上线运行，整个过程帮助王志瑞老师完成了很多的工作量，工作认真踏实，虚心好学，并帮助老师管理其它学生的学习与开发工作，为老师分担了很多的工作量。
